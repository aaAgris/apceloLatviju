# ApceloLatviju
Mājaslapa, kurā tās apmeklētāji var apskatīt Apceļo Latviju jaunākās aktualitātes, pakalpojumu piedāvājumus un pieteikties ceļojumu
piedāvājumiem. Mājaslapa veidota izmantojot HTML, CSS, Javascript un PHP.
